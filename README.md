## ⚙ Tech Stack
- Spring Boot 3.3.2
- Java 17
- STOMP Websocket
- MongoDB
- AWS EC2

<br>

## 💡 주요 기능
[🟠 POSTMAN Docs](https://documenter.getpostman.com/view/24157537/2sA3kYhec7)

- 채팅
- 채팅방
  - 채팅방 생성
  - 채팅방 상세 조회
  - 소설 첫문장 제시
  - 채팅방 참여하기(구독)
  - 채팅방 나가기
  - 참여중인 대기방 목록 조회 
- 소설책
  - 소설책 목록 조회
  - 소설책 조회
- 닉네임 중복확인
